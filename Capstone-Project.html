AI-Enhanced Profile Hosting Platform

Generation Singapore CSD04's Capstone Project

Description:

Developed a Flask-based web application hosted on Microsoft Azure, designed to showcase personal profiles while integrating modern AI-powered features. The platform combines web development, cloud services, automation, and payment integration to deliver a professional and scalable solution.

Key Highlights:

  • Built and deployed a Flask web application on Azure App Service.
  • Integrated AI functionality to provide smart, personalized responses.
  • Configured Azure Blob Storage for media and profile image management..
  • Set up Azure SQL Database (with SQLite fallback) for data storage.
  • Implemented Stripe payment gateway for donations/subscriptions.
  • Added secure user authentication with login/logout functionality.
  • Automated CI/CD pipeline using GitHub Actions for testing and deployment.
  • Enabled email notifications for user interactions.

Tech Stack:

  • Backend: Python (Flask)
  • Frontend: HTML, CSS, Bootstrap
  • Database: Azure SQL Database, SQLite (fallback) (PaaS)
  • Cloud Services: Azure App Service, Azure Blob Storage, Azure SQL
  • CI/CD: GitHub Actions
  • Payments: Stripe API
  • Other Tools: Git, VS Code, Azure CLI

Learning Outcome:

This project enhanced my expertise in:

  • Designing and deploying scalable cloud-based applications.
  • Using CI/CD pipelines for reliable and automated deployments.
  • Integrating cloud-native services such as storage and databases.
  • Leveraging third-party APIs (Stripe, AI services) to enrich applications.
  • Delivering an end-to-end solution from development to cloud deployment.
