PythonAnywhere Project

Generation Singapore CSD04's Mini Project

Description:

Developed and deployed a mini web application on PythonAnywhere as part of my Cloud & DevOps bootcamp. The project focused on learning the basics of Python web development and deployment on a cloud-based platform.

Key Highlights:

  • Built a Flask web application with simple functionality (e.g., user input, displaying dynamic results, or data storage).
  • Used HTML, CSS, and Bootstrap for frontend design.
  • Implemented server-side logic in Python (Flask framework).
  • Configured and deployed the application on PythonAnywhere, gaining hands-on experience with web hosting and domain setup.
  • Tested and maintained the live application to ensure accessibility from any device.

Tech Stack:

  • Backend: Python (Flask)
  • Frontend: HTML, CSS, Bootstrap
  • Hosting: PythonAnywhere (PaaS)
  • Other Tools: Git/GitHub for version control

Learning Outcome:

This project gave me practical exposure to:

  • Creating and running a Python-based web app.
  • Understanding the workflow of deployment to a hosting service.
  • Managing static files and dependencies in a real environment.
  • Preparing for more advanced cloud deployments (like my Azure project).
